# Melodia

## Sobre o projeto

O **Melodia** é um aplicativo fictício de músicas desenvolvido com foco em pessoas apaixonadas por música, descobertas musicais e experiências personalizadas. A proposta do projeto é criar uma interface moderna, organizada e responsiva para apresentar o aplicativo, destacando seus diferenciais, funcionalidades e formas de interação com o usuário.

Este projeto foi desenvolvido para o **Check-Point 05** da disciplina de **Front-end Design** do curso de **Engenharia de Software**. O desafio consiste na criação de uma **Landing Page** para o app Melodia, com estilo visual moderno, clean, cores vibrantes e elementos ligados ao universo musical.

---

## Objetivo

Desenvolver uma **Landing Page responsiva** para o aplicativo Melodia, utilizando:

- HTML5
- Tailwind CSS
- Font Awesome
- JavaScript

O projeto busca aplicar conceitos de estruturação de páginas, identidade visual, responsividade, organização de conteúdo e experiência do usuário, além de apresentar os principais diferenciais do aplicativo de forma clara e atrativa. 

---

## Proposta da aplicação

O **Melodia** foi idealizado como uma plataforma de música voltada para usuários que desejam ouvir suas músicas favoritas, descobrir novos artistas e montar playlists personalizadas de forma simples e intuitiva. Entre os diferenciais propostos para o aplicativo estão a **qualidade de som superior**, a **criação de playlists personalizadas**, a **descoberta de novos artistas** e uma **interface intuitiva**. 

A interface foi construída para oferecer uma navegação agradável, visual marcante e seções bem organizadas, reforçando a proposta de um app musical moderno e acessível. A página inicial apresenta uma hero section com chamada principal, uma área de benefícios, depoimentos de usuários e uma chamada para acesso à biblioteca musical.

---

## Funcionalidades

O projeto conta com as seguintes páginas e seções:

### Página Inicial (Home)
- Barra de navegação com logo, menu e botão de modo escuro
- Hero section com frase de destaque e botão de chamada para ação
- Seção de benefícios do aplicativo
- Área com depoimentos de usuários
- Chamada para criação de biblioteca musical
- Rodapé com contatos e redes sociais 

### Página App
- Estrutura separada para apresentar conteúdos relacionados ao aplicativo
- Seção com prévia de músicas disponíveis na plataforma
- Seção de download com opções para dispositivos mobile e desktop
- Rodapé padronizado e botão para alternância de tema claro/escuro 

### Página de Perfil
- Card com foto, nome de usuário e plano
- Estatísticas como playlists criadas, músicas curtidas e minutos ouvidos
- Seção “Sobre”
- Lista de artistas favoritos
- Lista de gêneros favoritos
- Playlists em destaque 

### Página de Suporte
- Formulário para envio de dúvidas, sugestões ou problemas
- FAQ com perguntas frequentes
- Manual do aplicativo com abas de navegação
- Mensagens de sucesso ao enviar ou limpar o formulário

## Tecnologias utilizadas

- **HTML5**
- **Tailwind CSS**
- **Font Awesome**
- **JavaScript**
- **Google Fonts**

O **Tailwind CSS** foi utilizado para construir uma interface moderna, responsiva e organizada com classes utilitárias.  
O **Font Awesome** foi usado para inserir ícones relacionados à navegação, benefícios e seções do sistema.  
O **JavaScript** foi aplicado em interações como a alternância entre modo claro e escuro e ações do formulário de suporte.  
Já o **Google Fonts** foi utilizado para complementar a identidade visual do projeto, deixando a interface mais harmoniosa e alinhada à proposta moderna do aplicativo. 

---

## Responsividade

Um dos principais objetivos do projeto foi garantir que a interface funcionasse bem em diferentes tamanhos de tela. Para isso, foram utilizados recursos de responsividade com classes utilitárias do Tailwind, organização com flex e grid e ajustes para diferentes breakpoints. As páginas apresentam estrutura adaptável para desktop e dispositivos menores, mantendo a legibilidade e a organização visual. 

---

## Estilização

A identidade visual do **Melodia** foi construída com foco em transmitir:

- modernidade
- dinamismo
- personalidade
- conexão com o universo musical

Foram utilizadas cores como **sangria**, **cosmic latte**, **cornflower blue** e tons escuros para o modo dark, criando contraste e um visual marcante. Essa paleta reforça a proposta vibrante e moderna do aplicativo.
